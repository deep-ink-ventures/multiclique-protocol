# Commons Library for MultiClique

[![Stellar](https://img.shields.io/badge/Stellar-v1.0-blue.svg)](https://www.stellar.org)
[![Rust](https://img.shields.io/badge/Rust-v1.73.0-orange.svg)](https://www.rust-lang.org)
[![License: Apache-2.0](https://img.shields.io/badge/License-Apache%202.0-yellow.svg)](https://opensource.org/licenses/Apache-2.0)

The `commons` library is a part of the MultiClique project, providing common functionality and reusable components for the MultiClique ecosystem. It aims to simplify development, reduce code duplication, and enhance maintainability across various parts of MultiClique.
